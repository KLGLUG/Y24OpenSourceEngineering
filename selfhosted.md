Kutt అనేది ఒక URL shortening service. దీని ద్వారా మీరు పొడుగు URLs ను చిన్న, షేర్ చేయదగిన URLs గా మార్చవచ్చు. Kutt URL clicks, analytics, custom aliases, password protection లాంటివి కూడా support చేస్తుంది.

Windows లో install చేయడానికి:

Node.js, Git, Redis install చేయండి.

Git ద్వారా Kutt repository clone చేసుకోండి: git clone https://github.com/thedevs-network/kutt.git

.env.example ను .env గా కాపీ చేసి, DATABASE_URL, SECRET_KEY లాంటి settings set చేయండి.

Kutt dependencies install చేయండి: npm install

Database migrate చేసి, Kutt run చేయండి: npm start
