# WikMD – ఓపెన్ సోర్స్ వికీ వ్యవస్థ

## వివరణ
**WikMD** ఒక ఓపెన్ సోర్స్ మార్క్‌డౌన్ ఆధారిత వికీ వ్యవస్థ.
ఇది వినియోగదారులు తమ డాక్యుమెంటేషన్‌ను స్థానికంగా (లోకల్‌గా) హోస్ట్ చేసుకొని సులభంగా నిర్వహించుకోవడానికి సహాయపడుతుంది.  
ఇందులో **రియల్ టైమ్ ప్రీవ్యూ**, **సులభమైన ఎడిటింగ్**, మరియు **తక్కువ స్థలాన్ని ఆక్రమించే వేగవంతమైన పనితీరు** వంటి లక్షణాలు ఉన్నాయి.  
వినియోగదారులు తమ స్వంత కంప్యూటర్‌లోనే ఒక documentation system సులభంగా సెట్ చేసుకోవచ్చు.

## ఇన్‌స్టాలేషన్ దశలు (Installation Steps)
1. ముందుగా **పైథాన్ (Python)** ను ఇన్‌స్టాల్ చేయండి.
   Python official site నుండి లేదా Ubuntu లో ఈ కమాండ్ వాడండి:
 sudo apt install python3
2. తర్వాత ఈ కమాండ్‌ని నడపండి:
 git clone https://github.com/veerasai06/wikmd.git
3. ఆ ఫోల్డర్‌లోకి వెళ్లండి:
 cd wikmd
4. తరువాత ఈ కమాండ్‌తో WikMD ను రన్ చేయండి:
 python3 wikmd.py
5. ఇప్పుడు మీ బ్రౌజర్‌లో ఈ లింక్ ఓపెన్ చేయండి:
 http://localhost:5000

ఇప్పుడు మీరు మీ స్వంత స్థానిక వికీ సిస్టమ్‌ను ఉపయోగించవచ్చు.

## ముగింపు
ఈ ప్రాజెక్ట్ ద్వారా నేను ఒక **సాధారణ, తేలికైన మరియు స్థానికంగా హోస్ట్ చేయగల వికీ సిస్టమ్**‌ను అభివృద్ధి చేశాను.
ఇది వ్యక్తిగత లేదా జట్టు డాక్యుమెంటేషన్ నిర్వహణకు చాలా ఉపయోగపడుతుంది.
ఈ ప్రాజెక్ట్‌లో భాగమయ్యి అనుభవాన్ని పొందడం ఒక అద్భుతమైన అవకాశంగా నిలిచింది.
