# ప్రాజెక్ట్ పేరు: Passky

## ప్రాజెక్ట్ వివరణ
Passky అనేది ఒక self-hosted password management tool.  
ఇది వ్యక్తిగత మరియు small team అవసరాలకు సరిపోయే సురక్షిత పాస్వర్డ్ స్టోరేజ్ సొల్యూషన్.

## ఫీచర్స్
1. పాస్వర్డ్ సేఫ్‌గా స్టోర్ చేయడం (Encryption).  
2. Multiple accounts management.  
3. సులభమైన search మరియు copy functionality.  
4. Team sharing support (optional).  
5. Self-hosted → డేటా local machine లోనే ఉంటుంది.

1. కోడ్ clone చేయండి:
   ```bash
   git clone https://github.com/your-username/passky.git

    Dependencies install చేయండి:

cd passky

    బ్రౌజర్ లో http://localhost:PORT ఓపెన్ చేయండి.
