# హోమ్ గ్యాలరీ
## Description
 హోమ్ గ్యాలరీ

హోమ్ గ్యాలరీ అనేది స్వీయ హోస్టింగ్ ఫోటో మరియు వీడియో గ్యాలరీ. ఇది మొబైల్ ఫ్రెండ్లీ, ట్యాగింగ్ మరియు AI ఆధారిత చిత్రం గుర్తింపు వంటి ఫీచర్లను కలిగి ఉంటుంది.

## లక్షణాలు
- స్వీయ హోస్టింగ్ ఫోటో/వీడియో గ్యాలరీ
- మొబైల్-ఫ్రెండ్లీ UI
- ట్యాగింగ్ మరియు శోధన
- AI ఆధారిత ఇమేజ్ గుర్తింపు



## Installation
Clone or download the repo from GitHub
git clone https://github.com/xemle/home-gallery.git
cd home-gallery
# Install required packages
npm install
npm run build
