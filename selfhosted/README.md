# My Self Hosted Project
# లింక్డింగ్ (Linkding)
## Description


లింక్డింగ్ అనేది స్వీయ-హోస్టెడ్ బుక్‌మార్క్ మేనేజర్. ఇది వేగంగా, తేలికగా మరియు డాకర్ ఉపయోగించి సులభంగా సెట్ చేయడానికి రూపొందించబడింది.

## లక్షణాలు
- సులభమైన బుక్‌మార్క్ నిర్వహణ
- డాకర్ సపోర్ట్
- ట్యాగ్‌ల ఆధారంగా సెర్చ్ మరియు ఫిల్టర్

## Installation
1)sudo apt update
2)sudo apt install python3 python3-pip nodejs npm make -y
3)git clone https://github.com/sissbruecker/linkding.git
4)cd linkding
5)make init
6)uv run manage.py createsuperuser --username=yourusername --email=you@example.com
7)make frontend
8)make serve
