# Baïkal – Self-Hosted CalDAV & CardDAV Server Documentation

## 1. Introduction
Baïkal is a lightweight, self-hosted CalDAV and CardDAV server used for synchronizing contacts and calendar events across devices. It gives complete privacy by storing all user data locally instead of relying on third-party cloud services like Google or iCloud.

This documentation explains how Baïkal was installed and configured on Ubuntu using Nginx, PHP-FPM, and SQLite.

---

## 2. Features of Baïkal
- CalDAV support for calendar syncing  
- CardDAV support for contact syncing  
- Lightweight and fast  
- Works inside local network  
- Private and secure — your data stays with you  
- Web admin panel  
- Supports SQLite and MySQL  

---

## 3. System Requirements
- Ubuntu Operating System  
- Nginx  
- PHP 8.4 + required extensions  
- SQLite  
- Git  
- Web browser
